# Greeting And Farewell Feature

Implement two simple string helpers.

## Requirements

- Export `buildGreeting(name: string): string`
- Export `buildFarewell(name: string): string`
- When the input is `Ada`, `buildGreeting` returns `Hello, Ada!`
- When the input is `Ada`, `buildFarewell` returns `Bye, Ada!`
- Do not modify the test files
